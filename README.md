# gravitons

Virtually massless CSS layout microlibrary

[![Build Status](https://travis-ci.org/jxnblk/gravitons.svg?branch=master)](https://travis-ci.org/jxnblk/gravitons)

http://jxnblk.com/gravitons

```sh
npm i -g gravitons
```

```sh
gravitons > base.css
```
